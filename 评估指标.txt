一、基础性指标
1. 准确率
概念：智能体输出结果中正确结果的比例，反映结果正确性。
计算方法：（正确输出结果数 / 总输出结果数）× 100%
2. 召回率
概念：智能体从目标数据集中成功识别的正确结果占所有真实正确结果的比例，反映结果完整性。
计算方法：（成功识别的正确结果数 / 数据集中所有真实正确结果数）× 100%
3. F1 值
概念：准确率与召回率的调和平均数，平衡正确性与完整性。
计算方法：（准确率 × 召回率 / 准确率 + 召回率）× 2
4. 任务完成率
概念：智能体在规定条件下完整完成目标任务的比例，反映任务执行最终效果。
计算方法：（成功完整完成任务的次数 / 总任务执行次数）× 100%
5. 平均耗时
概念：智能体完成单次 / 批量任务的平均时间消耗，反映执行效率。
计算方法：总任务执行时间 / 任务执行次数（单位：秒 / 毫秒）
二、通用化指标
1. 适应性指标
概念：智能体在非训练场景下的任务完成质量与训练场景的一致性，及对未知场景的容错与适配速度。
计算方法：
跨场景任务完成率偏差：非训练场景任务完成率 - 训练场景任务完成率
未知场景适配耗时：从首次接触未知场景到任务完成率稳定≥80% 的时间
场景覆盖度：（可有效适配的场景类型数 / 总测试场景类型数）× 100%
2. 鲁棒性指标
概念：智能体在异常条件（异常输入、环境波动、高并发）下的稳定运行能力。
计算方法：
异常输入错误率：（输入错误指令 / 噪声数据时的错误输出数 / 总输出数）× 100%
高并发稳定性：1 - （并发峰值时任务完成率 / 正常并发任务完成率）
环境波动容错率：1 - （算力 / 网络波动时的超时任务数 / 总任务数）× 100%
3. 可移植性指标
概念：智能体在不同硬件 / 软件环境中的部署成功率与适配成本，重点关注迁移后功能可用性。
计算方法：
跨环境部署成功率：（目标环境首次部署即正常运行的次数 / 总部署次数）× 100%
兼容性问题数：迁移后出现的硬件 / 软件兼容性错误次数
适配成本系数：迁移所需人力 / 时间成本 / 原部署成本
兼容性清单覆盖率：（工具生成的兼容性清单覆盖目标环境关键配置的数量 / 目标环境关键配置总数）× 100%
4. 协作效率指标
概念：智能体与其他智能体 / 人类协同完成任务时，信息交互准确性、协同速度与任务贡献度。
计算方法：
信息交互准确率：（智能体向协同对象传递的正确信息数 / 总传递信息数）× 100%
协同任务耗时差值：多主体协同完成任务的总时间 - 单主体完成同一任务的时间
协同贡献度：Σ（智能体完成的子任务占比 × 子任务重要性权重）
